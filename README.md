# team-conor

Claude Code용 팀 페르소나 설정 도구입니다.

## 사용법

```bash
npx team-conor
```

명령어를 실행하면 이름을 입력받고 다음 파일들이 생성됩니다:

```
CLAUDE.md
.conor/
├── persona/
│   ├── user.md          # 사용자 정보
│   ├── backend.md       # 빅토르 (BE 개발자)
│   ├── designer.md      # 마르코 (디자이너)
│   ├── frontend.md      # 유나 (FE 개발자)
│   ├── planner.md       # 스티브 (기획자)
│   └── scrum-master.md  # 엘런 (PM)
└── memory/              # 프로젝트 컨텍스트 저장소
```

## 팀원 소개

| 이름 | 역할 | 특징 |
|------|------|------|
| 스티브 | 기획자 | 잡스식 프로덕트 비저너리, "왜?"를 5번 물음 |
| 엘런 | PM | 머스크식 실행력, 블로커 제거 전문 |
| 마르코 | 디자이너 | 노먼 제자, 접근성 전도사 |
| 유나 | FE | React 코어팀 출신, 성능 최적화 덕후 |
| 빅토르 | BE | 25년차 아키텍트, 확장성 관점 |

## 사용 예시

특정 팀원에게 질문하면 해당 캐릭터로 응답합니다:

```
"유나, 이 컴포넌트 구조 어때?"
"빅토르, API 설계 리뷰 좀 해줘"
"엘런, 이번 스프린트 범위 확정하자"
```

다른 팀원이 관련 영역에서 자연스럽게 끼어들기도 합니다.
