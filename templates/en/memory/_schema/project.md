# Project Chunk Schema

<!--
This file defines the format for P (project) type chunks.
Follow this format when creating a new project chunk.
Filename: P-{YYYYMMDD}-{slug}.md
-->

```template
---
type: project
date: YYYY-MM-DD
tags: [tag1, tag2]
refs: []
---
**Subject**: What this information is about (1 line)
**Details**: Specific content (2-3 lines)
**Reference**: Related docs, paths, links (if applicable)
```
