# Decision Chunk Schema

<!--
This file defines the format for D (decision) type chunks.
Follow this format when creating a new decision chunk.
Filename: D-{YYYYMMDD}-{slug}.md
-->

```template
---
type: decision
date: YYYY-MM-DD
tags: [tag1, tag2]
refs: []
---
**Context**: Why this decision was needed (1 line)
**Decision**: What was chosen (1 line)
**Rationale**: Why this option was selected (1-2 lines)
**Alternatives**: Other options considered (1 line)
```
