# Learning Chunk Schema

<!--
This file defines the format for L (learning) type chunks.
Follow this format when creating a new learning chunk.
Filename: L-{YYYYMMDD}-{slug}.md
-->

```template
---
type: learning
date: YYYY-MM-DD
tags: [tag1, tag2]
refs: []
---
**Discovery**: What was found/learned (1 line)
**Cause**: Why this situation occurred (1 line)
**Solution**: How it was resolved (1-2 lines)
**Files**: Related file paths (if applicable)
```
